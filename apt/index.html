waypoint-stack apt repository

A GPG-signed, static apt repository of the pinned waypoint-stack digital-voice daemons for bookworm (armhf, arm64, amd64).

Install

Fetch the archive keyring and drop in a deb822 sources file, then install:

sudo curl -fsSL -o /usr/share/keyrings/waypoint-archive-keyring.gpg \
  https://kn4oqw.github.io/waypoint-stack/waypoint-archive-keyring.gpg

sudo curl -fsSL -o /etc/apt/sources.list.d/waypoint.sources \
  https://kn4oqw.github.io/waypoint-stack/waypoint.sources

sudo apt-get update
sudo apt-get install waypoint-stack

The sources file pins trust to the keyring via Signed-By:, so only packages signed by the Waypoint archive key are accepted — no repository-wide trust, no NO_PUBKEY workarounds.

Downgrade / rollback

The repository intentionally retains prior package versions in pool/. To pin a specific version:

apt-cache madison waypoint-mmdvmhost          # list available versions
sudo apt-get install waypoint-mmdvmhost=<version>

Trust roots

This repo is signed by a dedicated GPG archive key (waypoint-archive-keyring.gpg), used only for the apt Release/InRelease. It is separate from the RFC-0013 minisign release key, which remains the trust root for waypointd and reference data.
